# 商品期权分析文档 (非期货)

## 定位
剔除高波动的黑色系，聚焦与国际盘联动的金银有色，以及低波动的农产龙头。

## 策略性配置 ★★★☆☆

### 严选白名单
| 品种 | 代码 | 分类 |
|------|------|------|
| 黄金 | Au | 贵金属 |
| 白银 | Ag | 贵金属 |
| 沪铜 | Cu | 有色金属 |
| 沪铝 | Al | 有色金属 |
| 豆粕 | M | 农产品 |

### 核心策略
- 有色金属: 日历价差 (Calendar Spread)
- 黄金: 双卖收租

## 优势 (Pros)
- 国际定价锚定 (LME/COMEX)，抗内盘操纵
- 黄金自带避险属性
- 有色金属产业套保盘厚实

## 特定风险 (Specific Risk)
- 夜盘风险 (受美联储/非农数据影响跳空)
- 临近交割月强平规则依旧存在

## 反收割算法防线
- T-30 交割月红线
- 宏观事件日 (CPI/利率决议) 前强制减仓
