一、核心指标定义（4个维度，总分80分）

1. 收益指标（25分）

定义：年化收益率，衡量期权投资的回报潜力

公式：
年化收益 = (权利金收入 / 资金占用) × (365 / 到期天数) × 100%

资金占用：
• 卖出看跌：行权价 × 100（现金担保）
• 卖出看涨：股票价格 × 100（股票价值）

2. 安全指标（25分）

定义：行权概率，使用 Black-Scholes N(d2) 计算

公式：
$$P(\text{Assignment}) = N(d_2) = N\left( \frac{\ln(S/K) + (r - \frac{\sigma^2}{2})T}{\sigma\sqrt{T}} \right)$$

变量：S=股价, K=行权价, r=无风险利率(5%), σ=隐含波动率, T=到期时间(年)

3. 流动性指标（25分）

定义：复合流动性因子 = 40% 价差得分 + 60% 未平仓量(OI)得分

价差得分：价差率 = (卖价 - 买价) / 中间价
OI得分：基于未平仓量线性插值（10-500+张）

一票否决：OI < 10张 或 流动性因子 < 0.3

4. 成本指标（15分）

定义：买卖价差率，反映交易磨损成本

公式：
价差率 = (卖价 - 买价) / 中间价 × 100%

一票否决：价差率 > 10%

三、一票否决机制

以下任一条件不满足，期权直接得 0分，不予推荐：

1. 流动性过低

条件：流动性因子 < 0.3 或 未平仓量(OI) < 10张

理由：无法平仓，风险不可控

2. 收益过低

条件：年化收益 < 3%

理由：收益太低，不值得承担期权风险

3. 风险过高

条件：行权概率 > 75%

理由：Gamma风险急剧上升，流动性枯竭

4. 交易成本过高

条件：价差率 > 10%

理由：交易成本太高，侵蚀利润

五、流动性因子详细计算

复合流动性因子 = 40% × 价差得分 + 60% × OI得分

价差得分（40%权重）

• < 1%：1.0分

• 1-3%：0.8-1.0分（线性插值）

• 3-5%：0.5-0.8分（线性插值）

• 5-10%：0.2-0.5分（线性插值）

• > 10%：0.0分

价差率 = (卖价 - 买价) / 中间价

未平仓量(OI)得分（60%权重）

• ≥ 500张：1.0分

• 200-500张：0.8-0.95分（线性插值）

• 50-200张：0.6-0.8分（线性插值）

• 10-50张：0.3-0.6分（线性插值）

• < 10张：0.0分（一票否决）

• 数据缺失：0.3分（保守默认值）

说明：OI是判断流动性深度的"金标准"，特别是在盘前或交易清淡时，成交量可能为0，但OI能证明盘口有对手方存在。

六、完整计算示例

假设期权数据：

  • • 股票代码：AAPL，当前股价：$150.00
  • • 期权类型：PUT，行权价：$145.00
  • • 到期日：30天后，期权价格：$2.50
  • • 买价：$2.45，卖价：$2.55
  • • 隐含波动率：25%，未平仓量：300张

步骤1：计算行权概率

使用 Black-Scholes N(d2) 公式计算，PUT 行权概率 = 1 - N(d2) = 30.9%

步骤2：计算年化收益

权利金收入 = $2.50 × 100 = $250
资金占用 = $145 × 100 = $14,500
年化收益 = ($250 / $14,500) × (365 / 30) × 100% = 20.98%

步骤3：计算流动性因子

中间价 = ($2.45 + $2.55) / 2 = $2.50
价差率 = ($2.55 - $2.45) / $2.50 = 4%
价差得分：4% 在 3-5% 区间，线性插值 = 0.65分
OI得分：300张 在 200-500张 区间，线性插值 = 0.85分
流动性因子 = 0.4 × 0.65 + 0.6 × 0.85 = 0.77

步骤4：计算价差率

价差率 = (卖价 - 买价) / 中间价 × 100% = 4%

步骤5：一票否决检查

✅ 流动性 = 0.77 ≥ 0.3
✅ 年化收益 = 20.98% ≥ 3%
✅ 行权概率 = 30.9% ≤ 75%
✅ 价差率 = 4% ≤ 10%
✅ OI = 300 ≥ 10

步骤6：评分计算

  • 1. 年化收益：20.98% ≥ 15%，得 25分
  • 2. 行权概率：30.9% 在 25-35% 区间，线性插值 = 17.05分
  • 3. 流动性：0.77 在 0.6-0.8 区间，线性插值 = 23.25分
  • 4. 价差率：4% 在 3-5% 区间，线性插值 = 10.00分
  • 总分：25 + 17.05 + 23.25 + 10 = 75.30分
  • 推荐等级：≥ 60分，特别推荐，高亮显示 ✅

七、关键优化点（v2.1）

✅ 价差按比例计算

旧逻辑：固定金额（如 < $10）
新逻辑：价差率 = (Ask - Bid) / MidPrice
理由：$100的期权价差$0.5和$1的期权价差$0.5完全不同，比例更能反映交易磨损成本

✅ 风险阈值收紧

旧逻辑：行权概率 > 80% 一票否决
新逻辑：行权概率 > 75% 即视为过度风险
理由：实盘中，一旦概率超过75%，Gamma风险急剧上升，且此时往往已深度实值，流动性枯竭

✅ 流动性引入未平仓量

旧逻辑：仅看价差
新逻辑：复合因子 = 40% 价差得分 + 60% OI得分
理由：OI是判断流动性深度的"金标准"，特别是在盘前或交易清淡时，成交量可能为0，但OI能证明盘口有对手方存在

✅ 指标化呈现

将单纯的数值计算转化为标准化的 0.0 - 1.0 评分因子，并在文档中明确定义四个核心指标（收益、安全、流动性、成本）
